# Telegram-бот для магазина одежды

Бот показывает каталог одежды, избранное и дает покупателям связаться с вами: `@qyabape69`.

## Запуск

```bash
python bot.py
```

Дополнительные библиотеки устанавливать не нужно.

## Команды

- `/start` - меню
- `/catalog` - каталог
- `/available` - вещи в наличии
- `/favorites` - избранное
- `/contact` - связь с продавцом
- `/admin` - админка
- `/cancel` - отмена текущего действия
- `/set_start_sticker` - задать стикер для старта
- `/set_catalog_sticker` - задать стикер для каталога
- `/set_favorites_sticker` - задать стикер для избранного
- `/clear_stickers` - убрать сохраненные стикеры

## Как устроен каталог

При нажатии на **Каталог** бот показывает только товары из раздела каталога.

При нажатии на **В наличии** бот показывает только товары, которые вы добавили именно в раздел наличия.

После нажатия на название открывается карточка товара с фото, ценой, размерами, описанием, избранным и кнопкой связи.

В админке есть две отдельные кнопки: **Добавить в каталог** и **Добавить в наличие**. У существующего товара можно поменять раздел кнопкой **Перенести в каталог** или **Перенести в наличие**.

## Как добавить стикеры

Откройте админку или просто отправьте одну из команд:

```text
/set_start_sticker
/set_catalog_sticker
/set_favorites_sticker
```

После команды отправьте боту любой красивый Telegram-стикер. Бот сохранит его и будет показывать в выбранном разделе.
